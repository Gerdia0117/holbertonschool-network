# Networking Basics #0

This project covers fundamental networking concepts including:

- OSI Model
- LAN and WAN networks
- Internet and IP addressing
- TCP/UDP protocols
- Network tools and commands

## Learning Objectives

- Understanding the OSI model and its organization
- Differences between LAN and WAN networks
- IP addressing (IPv4 and IPv6)
- TCP vs UDP protocols
- Common network ports (SSH, HTTP, HTTPS)
- Using network diagnostic tools

## Requirements

- Ubuntu 22.04
- All Bash scripts must be executable
- Scripts must pass shellcheck
- First line: `#!/usr/bin/env bash`
- Second line: comment explaining the script
